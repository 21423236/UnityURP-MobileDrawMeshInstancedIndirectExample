 Why create this project?
 -------------
 To test DrawMeshInstancedIndirect API on mobile devices. And found that everything works very well even on weak GPU like adreno506.
 
 Note
 -------------
 This is a simplified example project, It doesnt not contain any GPU culling, just to test DrawMeshInstancedIndirect API on mobile .
